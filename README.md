A simple browser-based Tic Tac Toe game with selectable difficulty levels.

Features
- Play as X against an AI opponent.
- Difficulty selector: Easy (random), Medium (win/block), Hard (impossible to beat).
- Reset button to restart any time.

Quick Start
1. Open `index.html` in your browser.
2. Pick an AI difficulty from the dropdown.
3. Click any empty cell to place X; the AI plays O automatically.
4. Use 'Reset Game' to start over.
