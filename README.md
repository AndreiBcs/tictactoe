# Tic Tac Toe (Browser)

A simple browser-based Tic Tac Toe game with selectable AI difficulty levels.

## Features
- Play as X against an AI opponent.
- Difficulty selector: Easy (random), Medium (win/block), Hard (impossible to beat).
- Reset button to restart any time.
- Responsive grid layout with basic styling.

## Quick Start
1. Open `index.html` in your browser (no build step required).
2. Pick an AI difficulty from the dropdown.
3. Click any empty cell to place X; the AI plays O automatically.
4. Use **Reset Game** to start over.

## Notes
- The Hard AI uses minimax for optimal play; Medium adds basic win/block logic; Easy is random.
